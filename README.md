# Fatura Takip Sistemi

Fahri ve Metin için ortak fatura takip ve borçlandırma uygulaması.

## Kurulum Adımları

1. Railway.app'e GitHub ile üye ol.
2. Yeni proje oluştur, bu klasörü (repository'i) yükle.
3. Deploy et ve oluşan public URL'i kopyala.
4. Android telefonuna "SMS Gateway" uygulamasını Play Store'dan yükle ve çalıştır.
5. Uygulamayı açtığında ekranda lokal IP adresini göreceksin (örn: http://192.168.1.x:8080). 
   - Not: Telefon ile uygulamanın çalıştığı sunucunun aynı ağda olması gerekebilir veya port forwarding yapman gerekebilir. Lokal ağda çalıştırıyorsan direkt çalışacaktır.
6. Siteye gir -> `/ayarlar` sayfasına git ve SMS Gateway URL'ini gir.
7. Uygulamayı ilk kez açtığında `/setup` sayfası açılacaktır. Kullanıcıları oluştur.
8. Ayarlar sayfasından "Test SMS Gönder" butonuna basarak sistemi test et.
